作者: 陳子傑 M073040063
實作程式: python3
程式所需套件:  numpy, pandas, random, matplotlib, sklearn
使用方法:
開啟terminal, 確定在K-means_final資料夾內, 輸入以下指令:
python Main.py [演算法] [資料集] [迭代次數]
[演算法]: 有 ka (K-means Algorithm)和 gka (Genetic K-means Algorithm)兩種選擇
[資料集]: 有 iris 和 abalone 兩種選擇
[迭代次數]: 只要是整數都可以
程式執行完會用各種分群指標測量,其結果除了印出來外,也會輸出成文件檔存在report資料夾中做紀錄
